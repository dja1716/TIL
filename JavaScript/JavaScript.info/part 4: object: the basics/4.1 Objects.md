## delete property
```javascript
delete user.age;
```

## multiword propoerty
```javascript
let user = {
  name: "John",
  age: 30,
  "likes birds": true  // multiword property name must be quoted
};

// this would give a syntax error
user.likes birds = true
```
```javascript
let user = {};

// set
user["likes birds"] = true;

// get
alert(user["likes birds"]); // true

// delete
delete user["likes birds"];
```
